# Java-Multi-Thread-Learning


### 简介

关于Java多线程技术的学习中的例子汇总

```
com.sedion.mynawang
├── basic（多线程基础）
│   ├── DiscoverThread                              初识多线程
│   ├── PriorityThread                              线程优先级
│   ├── StatusThread                                线程状态
│   ├── DaemonThread                                守护线程
│   ├── SourceThread                                Thread线程源码阅读
│   ├── InterruptedThread                           线程中断
│   ├── SuspendThread                               暂停线程（过期）
│   ├── StopThread                                  停止线程（过期）
│   └── YieldThread                                 放弃当前线程CPU资源
├── advanced_synchrinozed（同步锁学习）
├── advanced_volatile（volatile关键字学习）
├── util（多线程工具）
│   ├── SleepUtils                                  休眠工具
```
